
## Setup commands
- `composer install`
- `npm install`
- `cp .env.example .env`
- `php artisan key:generate` -> After that Setup MySQL configuration
- `php artisan migrate:refresh --seed`
- `php artisan storage:link`


## Development Server
- `php artisan serve`
- `npm run dev`
